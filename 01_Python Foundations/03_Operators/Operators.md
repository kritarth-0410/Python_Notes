# 1. Arithmetic Operators

+  -  *  /  //  %  **

```python
a = 10
b = 3

print(a + b) # 13
print(a // b) # 3
print(a % b) # 1
```

Important:

- / gives float
- // gives floor division
- % gives remainder


# 2. Comparison Operators

== != > < >= <=

```python
print(5 == 5) # True
print(5 > 3) # True
```


# 3. Logical Operators

and or not

```python
x = 10
print(x > 5 and x < 20) # True
```


# 4. Assignment Operators

+= -= *= //=

```python
x = 3
x += 4
x *= 2
```


# 5. Membership Operators

in, not in

```python
print("a" in "apple")
```


# 6. Identity Operators

is, is not

```python
a = 10
b = 10
print(a is b)
```


# 7. Common Mistakes

- Using = instead of ==
- Confusing // and %
