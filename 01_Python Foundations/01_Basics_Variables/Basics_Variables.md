# 4. Variable Naming Rules

- Must start with letter or underscore
- Cannot start with number
- Case-sensitive
- No special characters except _

## Valid

```python
age = 20
_student = "AI"
```

## Invalid

```python
2name = "A"
my-name = 5
```
