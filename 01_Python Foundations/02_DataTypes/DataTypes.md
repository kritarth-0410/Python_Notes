# 2. Type Checking

```python

```


# 3. Type Casting 

```python
int("5") # 5
float(10) # 10.0
str(100) # "100"

int("abc")  # Error
```


# 4. Mutable vs Immutable

## Immutable:

- int
- float
- str
- tuple

## Mutable:

- list
- set
- dict

Example:

```python
s = "hello"
# s[0] = "H" # Error
```
